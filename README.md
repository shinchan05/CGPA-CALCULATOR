CGPA Calculator (C++)

 📌 Description
This is a C++ based CGPA calculator that computes CGPA using a credit-based system.  
It allows users to input subject names, grades, and credits, and displays a formatted report.

 🚀 Features
- Credit-based CGPA calculation
- Input validation for grades
- Supports multiple subjects
- Structured output display
- Menu-driven program

 🛠️ Technologies Used
- C++
- Standard Library

 ▶️ How to Run
```bash
g++ cgpa_calculator.cpp
./a.out
